# サブスク合計金額チェッカー（PWA）

毎月利用しているサブスクリプションサービスの
**月額合計・年額合計を簡単に可視化できるPWAアプリ**です。

「月5,000円くらいだから大丈夫」と思っていた支出が、
年換算でどれくらいになるのかを一目で確認できます。

---

## 🔗 公開URL
https://soranakatani2020-debug.github.io/subsc-checker/（あなたのGitHub PagesのURL）

※ スマートフォンではSafariで上記のリンクを開き右下の・・・から共有、その後「ホーム画面に追加」することで
ネイティブアプリのように利用できます。

---

## 📱 主な機能

- サブスク名・月額料金の追加
- サブスク一覧表示
- 月額合計金額の自動計算
- 年額合計金額の自動計算
- サブスクの削除
- データの永続化（localStorage）
- PWA対応（オフライン起動・ホーム画面追加）

---

## 🛠 使用技術

- HTML
- CSS

- localStorage
- PWA
  - Web App Manifest
  - Service Worker
- GitHub Pages（HTTPS配信）

※ フレームワークを使わず、基礎技術の理解を重視して実装しています。

## 💡 制作背景

サブスクリプションサービスが増える中で、
「毎月いくら使っているのか」「年間ではいくらになるのか」が
直感的に分かりづらいと感じたことがきっかけです。

特に **年額換算**を表示することで、
支出に対する気づきを得られるよう設計しました。

---

## 🎯 工夫した点

- 画面を1画面完結にして操作をシンプルにした
- 入力・削除後は即時再描画される設計
- localStorage を使い、ログイン不要で継続利用可能
- PWA化により、スマホアプリのようなUXを実現
